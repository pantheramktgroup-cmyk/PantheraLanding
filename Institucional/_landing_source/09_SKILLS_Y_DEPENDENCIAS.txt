## Dependencias npm recomendadas

Instalar:

npm install gsap @gsap/react lenis split-type

Tailwind según instalación Vite/React:

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

## Uso de GSAP

Usar:

- gsap
- ScrollTrigger
- useGSAP de @gsap/react

## Uso de Lenis

Usar Lenis para smooth scroll y conectarlo con ScrollTrigger.

## Uso de SplitType

Usar SplitType para animar headlines por línea/palabra.

## Importante

No usar demasiadas librerías extra.

No sumar Framer Motion salvo que sea estrictamente necesario. GSAP debe ser el sistema principal de animación.

No usar Three.js ni WebGL en esta primera versión. Podría verse impresionante, pero aumenta riesgo y peso. La prioridad es conversión, performance y una primera versión sólida.
