# PROMPT INICIAL PARA CLAUDE — LANDING PANTHERA GROUP

Actuá como un Senior Frontend Engineer + Creative Developer + Conversion Copywriter especializado en landings premium de alta conversión.

Objetivo: crear una landing completa para Panthera Group en React + Vite + Tailwind, con animaciones avanzadas usando GSAP y una estética premium, lujosa, minimalista y con inspiración renacentista.

La página tiene un único objetivo de conversión: que el visitante agende una llamada de diagnóstico en el calendario de GHL.

Reglas obligatorias:

- Todos los textos de la landing deben salir de un único archivo editable: src/content/landingCopy.js.
- Ningún título, subtítulo, CTA, FAQ, card o testimonio debe quedar hardcodeado dentro de los componentes.
- Todos los CTA deben llevar al calendario / sección de booking.
- No crear formularios propios. Usar el calendario embebido de GHL.
- No romper UTMs ni tracking.
- Mantener el script externo de GHL en index.html.
- Crear una primera versión muy completa, no un wireframe.
- Debe ser responsive, rápida y visualmente premium.
- Debe tener animaciones impactantes, pero no debe sentirse caótica ni pesada.
- Debe respetar prefers-reduced-motion.
- No usar estética genérica de SaaS, cards violetas, gradients baratos ni apariencia de template.
- Evitar promesas exageradas, claims no verificables o números rígidos como objetivos garantizados.
- El tono debe ser consultivo, directo, sofisticado, de autoridad y sin humo.

Stack requerido:

- React + Vite
- Tailwind CSS
- GSAP
- @gsap/react
- ScrollTrigger
- Lenis para smooth scroll
- SplitType para animación de titulares

Estructura esperada:

src/
  assets/
  components/
    layout/
    sections/
    ui/
  content/
    landingCopy.js
  hooks/
  lib/
  styles/
  App.jsx
  main.jsx

Entrega esperada:

- Proyecto funcional.
- Componentes separados por sección.
- landingCopy.js completo y bien ordenado.
- Animaciones encapsuladas y fáciles de ajustar.
- Código limpio, legible y mantenible.
- Tailwind configurado con paleta Panthera.
- Calendario GHL funcionando al final.
- CTAs con scroll suave al calendario.
- Navbar/sticky CTA opcional, sin invadir la experiencia.

No hagas preguntas. Ejecutá la mejor primera versión posible usando estos documentos como fuente principal.
