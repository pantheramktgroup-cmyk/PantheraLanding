# ESTRUCTURA DEL SITIO Y SECCIONES

Crear una web institucional multipágina para Panthera Group.

NAV PRINCIPAL

Items:
- Inicio
- Quiénes somos
- Servicios
- Resultados
- Contacto

CTA navbar:
- "Agendar diagnóstico"
- Link a `/contacto` o a una sección de booking si se integra luego.

HEADER

Estilo:
- fijo o sticky
- fondo transparente al inicio
- al hacer scroll: bg-black/70 con blur sutil
- logo Panthera
- links centrados o a la derecha
- CTA compacto
- responsive con menú mobile

PÁGINA 1 — INICIO `/`

Objetivo:
Presentar Panthera Group como una firma estratégica que construye sistemas comerciales para expertos y negocios high-ticket.

Secciones:

1. Hero institucional
- Full viewport o mínimo 90svh
- video o imagen cinematográfica de fondo
- overlay negro
- título grande: "Convertimos experiencia en un sistema comercial medible."
- subtítulo: "Panthera Group diseña, implementa y optimiza ecosistemas de captación y ventas para expertos, coaches, consultores y negocios high-ticket."
- CTA principal: "Agendar diagnóstico"
- CTA secundario: "Ver servicios"
- microcopy: "Estrategia · Captación · CRM · Automatización · Ventas"

2. Manifiesto breve
- Texto editorial: "No creemos en campañas sueltas. Construimos sistemas."
- Explicar que Panthera une estrategia, performance, automatización, seguimiento y operación comercial.

3. Servicios destacados
Cards:
- Arquitectura Comercial
- Captación y Performance
- CRM & Automatizaciones
- Optimización y Reporting

4. Sistema Panthera APEX
Mostrar fases:
01 Diagnóstico
02 Auditoría
03 Arquitectura
04 Construcción
05 Activación
06 Optimización
07 Escalabilidad

5. Resultados / Casos
Si hay testimonios reales disponibles, usar nombres reales existentes:
- Gastón Hendlin
- Laura Sanchez
- Lucas Casalins
- José Navas
- Hilda Arjona
Si faltan assets, dejar placeholders claros.

6. CTA final
- "Si tu negocio ya vende, pero todavía depende demasiado de vos, hablemos."
- Botón: "Agendar diagnóstico"

PÁGINA 2 — QUIÉNES SOMOS `/quienes-somos`

Objetivo:
Mostrar autoridad, visión y filosofía de Panthera.

Secciones:

1. Hero
- "No somos una agencia. Somos el equipo que construye tu sistema comercial."
- Subtítulo sobre operar desde estrategia, ejecución y mejora continua.

2. Historia / Origen
Texto real parcial:
"Panthera nace para resolver un problema común en expertos y negocios high-ticket: tener valor real, pero no contar con un sistema predecible para atraer, filtrar y convertir oportunidades."

3. Filosofía
Bloques:
- Estrategia antes que ejecución
- Sistema antes que campaña
- Datos antes que intuición
- Calidad antes que volumen

4. Fundadores / Equipo
Usar fotos reales si existen:
- fundadores.webp
- fundador_1.webp
- fundador_2.webp
Si no existen en el nuevo proyecto, usar placeholders y lorem ipsum.

5. CTA
"Conocé cómo podríamos estructurar tu sistema comercial."

PÁGINA 3 — SERVICIOS `/servicios`

Objetivo:
Explicar qué hace Panthera y para quién.

Secciones:

1. Hero
- "Servicios diseñados para crear un sistema, no solo una campaña."

2. Servicios principales
Cards o secciones:
- Diagnóstico comercial
- Arquitectura de funnel
- Landing pages y activos de conversión
- Captación paga y orgánica
- CRM, automatizaciones y seguimiento
- Dashboards y reporting
- Optimización comercial
- Consultoría estratégica

3. Proceso de trabajo
Fases APEX.

4. Para quién es / Para quién no es
Para:
- expertos con oferta validada
- coaches/consultores/mentores high-ticket
- empresas que necesitan ordenar captación y ventas
- negocios que ya venden pero no tienen previsibilidad

No es para:
- negocios sin oferta validada
- personas buscando soluciones mágicas
- empresas que solo quieren "hacer anuncios" sin proceso comercial

5. CTA

PÁGINA 4 — RESULTADOS `/resultados`

Objetivo:
Mostrar casos, testimonios y prueba social.

Secciones:
- Hero: "Resultados que nacen de sistemas, no de casualidad."
- Grid de testimonios
- Video testimonials si hay assets
- Métricas si son reales
- Si no hay datos verificables, usar placeholders y no inventar cifras.

PÁGINA 5 — CONTACTO `/contacto`

Objetivo:
Convertir a reunión.

Secciones:
- Hero simple
- Formulario o embed calendario
- Datos de contacto
- FAQ breve
- Texto:
  "Contanos en qué etapa está tu negocio y evaluamos si Panthera puede ayudarte a construir un sistema comercial más predecible."

FOOTER

Incluir:
- Logo
- navegación
- redes si existen
- copyright
- texto breve:
  "Panthera Group — Sistemas comerciales para expertos y negocios high-ticket."
