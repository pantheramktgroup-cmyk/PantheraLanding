# ASSETS RECOMENDADOS Y CÓMO USARLOS

El ZIP de Panthera contiene logos, tipografías, fotos de fundadores e imágenes renacentistas.

## Carpeta recomendada dentro del proyecto

Copiar assets seleccionados a:

public/assets/panthera/
  logos/
  renaissance/
  founders/

O, si preferís importar desde src:

src/assets/panthera/
  logos/
  renaissance/
  founders/

Para Vite/React, public suele ser más simple para una primera versión.

## Imágenes renacentistas

Usar preferentemente las versiones SIN texto incrustado, para que no compitan con el copy.

Desde el ZIP original:

Panthera/Renacentismo/2.png
Uso recomendado:
Hero o fondo de statement. Es un rostro renacentista cercano, ideal para imagen principal con overlay oscuro.

Panthera/Renacentismo/4.png
Uso recomendado:
Sección video/manifiesto o transición. Tiene manos y composición ceremonial, útil para un bloque más emocional.

Panthera/Renacentismo/6.png
Uso recomendado:
Fondo de sección “Lo que se ordena dentro del sistema” o comparativa. Es más abstracta y no compite tanto.

Panthera/Renacentismo/8.png
Uso recomendado:
Sobre Panthera o problema central. Tiene figura humana, tono oscuro y presencia premium.

Evitar como fondo principal:

Panthera/Renacentismo/1.png

Tienen texto Panthera incrustado. Pueden usarse como referencia visual, pero no conviene poner texto encima de texto.

## Tipografías

El ZIP incluye HelveticaNowDisplay.

Usar si el proyecto puede cargar fuentes locales.

No compartir ni redistribuir las fuentes fuera del proyecto.

Si no se configuran localmente en la primera versión, usar fallback:

Inter, Helvetica Neue, Arial, sans-serif.

## Video de Manuel

Cuando exista archivo final, colocarlo en:

public/assets/panthera/videos/manuel-vsl.mp4

- poster oscuro.
